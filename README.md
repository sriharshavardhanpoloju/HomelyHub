# HomelyHub – Hotel Booking Web Application

HomelyHub is a full-stack hotel booking platform developed using the MERN stack. It enables users to browse hotels, make reservations, and manage bookings through a responsive web interface.

## Features

- User Authentication
- Hotel Listings
- Room Booking
- Booking Management
- Responsive UI
- RESTful APIs

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
